**Задание:**

Доступ к pdf потерян, известно только то, что пароль это дата выхода (в формате dd.mm.YYYY) популярной песни.

---
**Решение:**

У нас есть запароленный PDF-файл. Название и описание задания прямо намекают на единственное верное решение – перебор паролей.

Решение №1 – брутфорс при помощи hashcat.

Первым делом формируем словарь с паролями:

```shell
python3 - << 'EOF' > dates.txt                         
from datetime import date, timedelta

start = date(1950, 1, 1)
end = date(2026, 12, 31)

d = start
while d <= end:
    print(d.strftime("%d.%m.%Y"))
    d += timedelta(days=1)
EOF
```

Теперь нам нужно извлечь хэш пароля из PDF:

```shell
pdf2john flag.pdf | sed 's/^[^:]*://' > pdf.hash

cat pdf.hash           
$pdf$5*6*256*-4*1*16*fb24aed63e83dd5f5f6257da4a2ca29e*48*acce720893cadb7971373ad09c659a8b1348e9a01cc6f4f12854de622d51d8d0e1afbff48f2d7d739bb3851f8e262ae3*48*c12c55efe45dc7eca8838d71302698b6812d9a13911a802cc476b10180e12048ef49eeab8b2dabe7b07519be89d57c9b*32*a942e4a98e63c8577ed1c753975f186835e663778a7617ea0c6f53320bafd8e7*32*b1edf985d662a374252a6a6e178950c199f8718f27b266d95bda2509066cc92d
```

По параметрам шифрования ясно, что это PDF 1.7 Level 8 (Acrobat 10-11), поэтому нам нужно будет указать следующий тип хэша:

```
10700 | PDF 1.7 Level 8 (Acrobat 10 - 11)
```

Полная команда для перебора:

```
hashcat -a 0 -m 10700 pdf.hash dates.txt
```

Ждём, когда hashcat подберёт пароль, открываем PDF и получаем флаг.

Решение №2 – скрипт, который перебирает даты через `qpdf`:

```shell
python3 - << 'EOF'
import subprocess
from datetime import date, timedelta

pdf = "flag.pdf"

start = date(1960, 1, 1)
end = date(2026, 4, 25)

d = start
while d <= end:
    pwd = d.strftime("%d.%m.%Y")

    r = subprocess.run(
        ["qpdf", f"--password={pwd}", "--check", pdf],
        stdout=subprocess.DEVNULL,
        stderr=subprocess.DEVNULL
    )

    if r.returncode == 0:
        print(f"\nFOUND: {pwd}")
        break

    d += timedelta(days=1)
else:
    print("Password not found")
EOF
```
