**Задание:**

Очень хочется получить весть флаг целиком, но как бы я ни пытался, получаю только первую его часть, я даже пробовал открывать адрес через разные браузеры – результат один. Попробуй, может у тебя получится?

Адрес: http://31.207.77.216:4011

---
**Решение:**

Делаем запросы с разными `User-Agent`, передавая куку, и получаем флаг:

```bash
curl -c cookies.txt http://31.207.77.216:4011
curl -b cookies.txt -H "User-Agent: Mozilla/5.0 (iPhone)" http://31.207.77.216:4011
curl -b cookies.txt -H "User-Agent: Googlebot" http://31.207.77.216:4011
curl -b cookies.txt -H "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36" http://31.207.77.216:4011
curl -b cookies.txt -H "User-Agent: curl/7.68.0" http://31.207.77.216:4011
```
