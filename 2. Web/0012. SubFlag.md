**Задание:**

Один из поддоменов bug-makers.ru скрывает флаг, найдите поддомен и сам флаг.

---
**Решение:**

Ищем поддомены:

```bash
subfinder -d  bug-makers.ru -o domen_3.6.txt
```

Смотрим TXT-записи и находим флаг:

```bash
cat domen_3.6.txt | xargs -I {} sh -c 'echo "{}:" && dig +short TXT {}'
subdomain-for-the-task.bug-makers.ru:
"Flag: BugCTF{XXXXXXXXXXXXXXX}"
alex.bug-makers.ru:
"v=spf1 redirect=beget.com"
ctf.bug-makers.ru:
"v=spf1 redirect=beget.com"
```
