**Задание:**

Вокруг света за 80 дней минуту! Путешествовать всегда хорошо! Сувениры там всякие привозишь, диковинки всякие наблюдаешь. Но не всегда получается выкроить время :( Так что отправимся в путешествие виртуальное, а вместо сувениров - куски флагов! Пока все не объедешь, флаг не получишь!

Адрес: http://31.207.77.216:4018

---
**Решение:**

Делаем запросы с разными прокси (брал тут – https://cdn.jsdelivr.net/gh/proxifly/free-proxy-list@main/proxies/protocols/socks5/data.txt), чтобы получить все части флага:

```python
import requests
import random
import time
from urllib.parse import urlparse

TARGET = "http://31.207.77.216:4018"

PROXY_FILE = "proxies.txt"

with open(PROXY_FILE, "r") as f:
    PROXIES = [line.strip() for line in f if line.strip()]

session = requests.Session()

flag_parts = set()

def proxy_dict(p):
    return {
        "http": p,
        "https": p
    }

print("[*] Starting rotation...")

while True:
    p = random.choice(PROXIES)

    print(f"\n[*] Using proxy: {p}")

    try:
        resp = session.get(
            TARGET,
            proxies=proxy_dict(p),
            timeout=8
        )
        text = resp.text.strip()

        if "flag" in text.lower():
            print("\n[+] FLAG PART FOUND!")
            print(text)
            flag_parts.add(text)

        else:
            print("[OK] Response OK, but no new flag part.")

    except Exception as e:
        print(f"[!] Proxy failed: {p} | {e}")

    print(f"Collected flag parts: {len(flag_parts)}")
    time.sleep(0.5)
```

В ответе от сервера получаем части флага – всего 10 частей, собираем все, останавливаем скрипт и получаем флаг.
