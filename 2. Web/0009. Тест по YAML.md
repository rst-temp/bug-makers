**Задание:**

Чтобы получить флаг, нужно набрать 1337 баллов.

Адрес задачи: http://31.207.77.216:4002

---
**Решение:**

Для получения флага нужно сделать YAML, который по версии 1.1 даст `score = 1337`, а по версии 1.2 даст другой результат, чтобы условие `score_1 !== score_2` сработало.

Решение – `sexagesimal integer (формат H:M:S)`. В YAML 1.1 запись вида:

```text
0:22:17
```

Интерпретируется как:

```text
0 часов * 3600 + 22 минуты * 60 + 17 секунд = 1337
```

В YAML 1.2 это НЕ число → просто строка `"0:22:17"`.

Значит: `score_1 = 1337` → `score_2 = "0:22:17"` → `score_1 !== score_2` → сервер возьмёт `score = score_1 = 1337` → получим флаг:

```shell
curl -X POST http://31.207.77.216:4002/submit \
  --data-urlencode "result=result:
  - score: 0:22:17"

{"pass":true,"flag":"BugCTF{XXXXXXXXXXXXXXX}"} 
```
