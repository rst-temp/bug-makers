**Задание:**

Жемчужину скрывает только одна ракушка из десятков тысяч.

Адрес: http://155.212.186.222

---
**Решение:**

Задание намекает, что нужно найти нужный порт. Скрипт для поиска флага:

```python
import requests
import re
import sys
from multiprocessing import Pool

def scan(p):
    try:
        r = requests.get(f'http://155.212.186.222:{p}', timeout=3)
        if r.status_code == 200:
            m = re.search(r'BugCTF\{.*?\}', r.text)
            if m:
                print(f'Port {p}: {m.group()}')
                sys.exit(0)
    except:
        pass

if __name__ == '__main__':
    with Pool(200) as ex:
        ex.map(scan, range(1, 65536))
```
